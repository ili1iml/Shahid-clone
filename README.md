# Shahid-clone
A front-end recreation of the Shahid streaming platform interface.
This project focuses on building a clean, modern, and responsive UI inspired by real-world streaming services.

# ⭐ Features
Modern and responsive layout
Hero section, categories, posters, and footer
Smooth user experience and clean visual design
Organized structure for easy expansion
Ready to integrate with APIs or backend in the future

# 🛠️ Technologies Used
HTML5
CSS3
JavaScript

# 🎯 Project Goals
Improve front-end development skills
Practice UI/UX replication from real platforms
Build strong portfolio-ready projects
Explore layout design and responsive techniques
